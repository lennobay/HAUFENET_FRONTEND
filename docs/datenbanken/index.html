Datenbanken Grundlagen

Status: In creation ( last edit: 17:21, 06.10.2026)

Datenbanken benutzten wir jeden Tag auf diesem Planet, so lange man das Internet benutzt. Hierbei braucht man die sowohl in Discord, Instagram, Teams oder in jeglichen anderen Applikationen. Somit stellen diese einen wichtigen Teil des gesamten Internetes dar.

Was ist eine Datenbank?

Eine Datenbank ermöglicht es Daten in einer strukturierten Form zu speichern und deren Zugriff so einfach wie möglich zu gestalten. Um diese zu strukturieren werden dabei verschiedenste "Modelle" benutzt. Eines davon ist das Modell der Relationalen Datenbank, mit welchem wir uns beschäftigen werden. Dabei gibt es auch andere Beispiele wie Beispielsweise Redis. Diese unterstützt hierbei verschiedenste Arten die Daten zu speichern, ob als Hash, Tabelle oder JSON. Doch es vereint ein einheitliches system in welchem Datensätze mit einem Key Value Prinzip gespeichert werden. MongoDB verfolgt hierbei eher in teilen ein similares System aber unterstützt nur ein JSON-artiges Format für desen Datensatzen. Die besten Beispiele für die Relationalen Datenbanksystemen sind hierbei Postgresql ( mein Favorit), MySQL sowie Mariadb. Die letzteren haben aber zurzeit exstenziäle Probleme.

Aufbauprinzip eines Datenbanksystemes

Ein Datenbanksystem besteht dabei aus mehrern Teilen.
